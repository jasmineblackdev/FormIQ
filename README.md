##  Overview

FormIQ allows users to record their exercises, receive AI-driven form feedback, track workouts, and review progress over time.  
Unlike traditional fitness apps that focus only on reps and weight, **FormIQ prioritizes movement quality and technique**.

The app adapts to:
- Beginners learning correct form
- Intermediate lifters refining technique
- Bodybuilders optimizing execution
- Powerlifters improving mechanics
- General gym-goers training safely

##  User Experience

### Typical Workout Flow
1. User opens the app
2. Selects a workout or exercise
3. Views demo (optional)
4. Records a set or logs manually
5. AI analyzes form
6. Feedback is presented
7. User logs the set
8. Progress and videos are saved automatically

Users can train **with or without a predefined workout plan**.
